# 字段选择器修复说明

## 问题分析

### 发现的问题

1. **初始化时机问题**：
   - 字段选择器只在"项目总表"标签页被点击时才初始化
   - 如果用户没有点击该标签页，字段选择器就不会被初始化
   - 导致字段选择按钮无法显示

2. **应用功能限制**：
   - 字段选择器的应用功能只能在"项目总表"标签页中使用
   - 在其他标签页中点击"应用字段选择"会失败
   - 缺乏用户友好的提示信息

3. **用户体验问题**：
   - 用户不知道字段选择器需要在特定标签页中使用
   - 缺乏明确的使用说明

## 修复方案

### 1. 改进初始化时机

**修改前**：
```javascript
// 只在项目总表标签页点击时初始化
const summaryTabBtn = document.getElementById('summary-tab');
if (summaryTabBtn) {
    summaryTabBtn.addEventListener('click', function() {
        // 初始化字段选择器
        initializeColumnSelector();
    });
}
```

**修改后**：
```javascript
// 页面加载完成后立即尝试初始化
setTimeout(() => {
    console.log('立即尝试初始化字段选择器...');
    const success = initializeColumnSelector();
    if (!success) {
        console.log('立即初始化失败，将在1秒后重试...');
        setTimeout(() => {
            initializeColumnSelector();
        }, 1000);
    }
}, 100);
```

### 2. 增强应用功能

**修改前**：
```javascript
function applyColumnSelection() {
    if (!summaryTable) {
        showToast('表格未初始化，无法应用字段选择', 'error');
        return;
    }
    // ... 应用逻辑

```

**修改后**：
```javascript
function applyColumnSelection() {
    // 检查是否在项目总表标签页
    const summaryTab = document.getElementById('summary-tab');
    const summaryContent = document.getElementById('summary');
    
    if (!summaryTab || !summaryContent || !summaryContent.classList.contains('active')) {
        showToast('请先切换到"项目总表"标签页，然后应用字段选择', 'warning');
        return;
    }
    
    if (!summaryTable) {
        showToast('表格未初始化，正在尝试初始化...', 'info');
        // 尝试初始化表格
        if (typeof loadProjectSummary === 'function') {
            loadProjectSummary();
            setTimeout(() => {
                if (summaryTable) {
                    applyColumnSelection();
                } else {
                    showToast('表格初始化失败，无法应用字段选择', 'error');
                }
            }, 1000);
        } else {
            showToast('表格未初始化，无法应用字段选择', 'error');
        }
        return;
    }
    // ... 应用逻辑
}
```

## 使用说明

### 基本使用流程

1. **打开字段选择器**：
   - 点击"字段选择器"标题右侧的箭头按钮
   - 展开字段选择面板

2. **选择字段**：
   - 在左侧面板中点击字段按钮进行选择/取消选择
   - 蓝色按钮表示已选择，灰色按钮表示未选择

3. **应用选择**：
   - 切换到"项目总表"标签页
   - 点击"应用字段选择"按钮
   - 系统会重新加载表格并显示选中的字段

4. **快捷操作**：
   - "全选字段"：选择所有字段
   - "清空选择"：取消所有字段选择
   - "手动初始化"：重新初始化字段选择器

### 注意事项

- 字段选择器只对"项目总表"标签页有效
- 如果表格未初始化，系统会自动尝试初始化
- 建议在应用字段选择前先切换到"项目总表"标签页
