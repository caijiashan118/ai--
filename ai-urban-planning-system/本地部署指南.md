# AI城建系统 - 本地部署指南

## 🎯 系统概述

AI城建系统是一个基于Flask的智能城市建设管理系统，提供项目信息管理、资金安排、数据分析和可视化功能。

## 📋 系统要求

- **操作系统**: Windows 10/11, macOS 10.14+, Linux Ubuntu 18.04+
- **Python版本**: Python 3.7 或更高版本
- **内存**: 至少 2GB RAM
- **存储空间**: 至少 500MB 可用空间

## 🚀 快速部署

### 方法一：一键启动（推荐）

#### Windows用户
1. 双击运行 `start.bat`
2. 等待依赖安装完成
3. 浏览器访问 http://localhost:8080

#### macOS/Linux用户
1. 打开终端，进入项目目录
2. 运行 `./start.sh`
3. 浏览器访问 http://localhost:8080

### 方法二：手动部署

#### 1. 检查Python环境
```bash
python3 --version
# 或
python --version
```

#### 2. 安装依赖包
```bash
pip3 install -r requirements.txt
# 或
pip install -r requirements.txt
```

#### 3. 启动应用
```bash
python3 app.py
# 或
python app.py
```

## 📊 功能模块

### 1. 基础数据模块
- **项目总表**: 汇总所有项目信息，支持多条件筛选
- **项目基本信息表**: 管理项目基础数据
- **项目资金安排表**: 管理资金分配信息
- **数据导入导出**: 支持Excel格式
- **字段选择器**: 自定义显示字段

### 2. 智能图表模块
- **实时监控**: 项目状态实时监控
- **数据分析**: 项目进度和资金执行分析
- **风险预警**: 自动识别项目风险
- **可视化图表**: 多种图表类型展示

### 3. 用户管理模块
- **用户管理**: 创建、编辑、删除用户
- **角色权限**: 基于角色的权限管理
- **操作日志**: 完整的操作审计日志

## 🗄️ 数据库配置

### 默认配置（SQLite）
系统默认使用SQLite数据库，无需额外配置：
- 数据库文件: `instance/ai_urban_system.db`
- 自动创建表结构
- 支持数据备份和恢复

### MySQL配置（可选）
如需使用MySQL数据库，请运行：
```bash

```

## 🔧 配置说明

## 🔍 故障排除

### 常见问题

#### 1. 端口被占用
**错误**: `Address already in use`
**解决方案**:
- 修改 `app.py` 中的端口号
- 或停止占用端口的其他程序

#### 2. 依赖安装失败
**错误**: `pip install` 失败
**解决方案**:
```bash
# 升级pip
pip install --upgrade pip
# 重新安装依赖
pip install -r requirements.txt --force-reinstall
```

#### 3. 数据库连接失败
**错误**: 数据库相关错误
**解决方案**:
- 检查 `instance/` 目录权限
- 删除数据库文件重新创建
- 检查磁盘空间

#### 4. 权限错误
**错误**: 文件权限相关错误
**解决方案**:
```bash
# Linux/Mac
chmod +x start.sh
chmod 755 instance/
```

日志文件保存在 `logs/` 目录下。

## 🚀 生产环境部署

```bash

gunicorn -w 4 -b 0.0.0.0:8080 app:app
```

### 使用Nginx反向代理
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
