# AI城建系统

一个基于Flask的智能城市建设管理系统，提供项目信息管理、资金安排、数据分析和可视化功能。

## 🚀 快速开始

### 方法一：自动安装（推荐）

```bash
python install.py
```

### 方法二：手动安装

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动应用**
   ```bash
   python app.py
   ```

3. **访问应用**
   打开浏览器访问：http://localhost:8080

## 📋 系统功能

### 基础数据模块
- **项目总表**：汇总所有项目信息，支持多条件筛选
- **项目基本信息表**：管理项目基础数据
- **项目资金安排表**：管理资金分配信息
- **字段选择器**：自定义显示字段
- **数据导出**：支持Excel格式导出

### 智能图表模块
- **实时监控**：项目状态实时监控
- **数据分析**：项目进度和资金执行分析
- **风险预警**：自动识别项目风险
- **可视化图表**：多种图表类型展示

## 🛠️ 技术特性

- **响应式设计**：支持各种屏幕尺寸
- **数据一致性**：自动确保数据完整性
- **高性能**：优化的数据库查询
- **易扩展**：模块化设计，易于扩展

## 🔧 配置说明

### 数据库配置
- 默认使用SQLite数据库
- 数据库文件：`instance/ai_urban_system.db`
- 支持MySQL（需修改配置）

### 端口配置
- 默认端口：8080
- 可在 `app.py` 中修改

## 🔍 故障排除

### 常见问题

1. **端口被占用**
   ```bash
   # 修改端口
   python app.py --port 8081
   ```

2. **依赖安装失败**
   ```bash
   # 升级pip
   pip install --upgrade pip
   # 重新安装
   pip install -r requirements.txt --force-reinstall
   ```

3. **数据库错误**
   ```bash
   # 删除数据库文件重新创建
   rm instance/ai_urban_system.db
   python app.py
   ```
