# 🚀 AI城建系统 - 快速启动

## 一键启动

### Windows用户
```bash
双击运行 start.bat
```

### macOS/Linux用户
```bash
./start.sh
```

## 访问系统

- **地址**: http://localhost:8082
- **管理员账号**: admin / admin123

## 主要功能

✅ **基础数据管理**
- 项目信息管理
- 资金安排管理
- 数据导入导出
- 多条件筛选

✅ **智能图表分析**
- 实时数据监控
- 项目进度分析
- 风险预警系统

✅ **用户权限管理**
- 多角色权限控制
- 操作日志审计

## 系统要求

- Python 3.7+
- 2GB+ RAM
- 500MB+ 存储空间

## 故障排除

**端口被占用**: 修改 `app.py` 中的端口号
**依赖安装失败**: 运行 `pip install -r requirements.txt --force-reinstall`
**权限错误**: 运行 `chmod +x start.sh`

详细说明请查看 `本地部署指南.md`
