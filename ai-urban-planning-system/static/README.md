# 静态资源文件

本目录包含系统运行所需的静态资源文件，包括：

## CSS文件
- bootstrap.min.css - Bootstrap 5.1.3 样式框架
- font-awesome.min.css - Font Awesome 6.0.0 图标库
- dataTables.bootstrap5.min.css - DataTables Bootstrap5 集成样式

## JavaScript文件
- jquery.min.js - jQuery 3.6.0 库
- bootstrap.bundle.min.js - Bootstrap 5.1.3 完整包
- jquery.dataTables.min.js - DataTables 1.13.4 核心库
- dataTables.bootstrap5.min.js - DataTables Bootstrap5 集成
- plotly.min.js - Plotly.js 2.27.0 图表库
- xlsx.full.min.js - SheetJS 0.18.5 Excel处理库

## 更新说明
如需更新这些文件，请运行：
```bash
python download_assets.py
```
