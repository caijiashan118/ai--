## 🔧 故障排除

### 常见问题

1. **端口被占用**
   ```bash
   # 查看端口占用
   lsof -i :8080
   # 杀死占用进程
   kill -9 <PID>
   ```

2. **依赖包问题**
   ```bash
   # 重新安装依赖
   pip3 install -r requirements.txt --force-reinstall
   ```

3. **数据库问题**
   ```bash
   # 删除数据库重新创建
   rm -rf instance/
   python3 app.py
   ```
