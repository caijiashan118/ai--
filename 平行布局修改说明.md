# 筛选器和字段选择器平行布局修改说明

## 修改概述

将原本垂直排列的筛选器和字段选择器改为平行关系（水平排列），并进一步拉伸水平布局，提升用户体验和界面利用率。

## 修改内容

### 1. 布局结构调整

**修改前：**
```html
<!-- 筛选器 -->
<div class="card mb-3" id="filterCard">
    <!-- 筛选器内容 -->
</div>

<!-- 字段选择器 -->
<div class="row mt-3 mb-3">
    <div class="col-12">
        <div class="card border-info">
            <!-- 字段选择器内容 -->
        </div>
    </div>
</div>
```

**修改后：**
```html
<!-- 筛选器和字段选择器平行布局 -->
<div class="row mb-3">
    <!-- 筛选器 -->
    <div class="col-xl-9 col-lg-8">
        <div class="card" id="filterCard">
            <!-- 筛选器内容 -->
        </div>
    </div>
    
    <!-- 字段选择器 -->
    <div class="col-xl-3 col-lg-4">
        <div class="card border-info h-100">
            <!-- 字段选择器内容 -->
        </div>
    </div>
</div>
```

### 2. 响应式布局（拉伸优化）

- **超大屏幕（≥1200px）**：筛选器占 9/12 宽度，字段选择器占 3/12 宽度
- **大屏幕（≥992px）**：筛选器占 8/12 宽度，字段选择器占 4/12 宽度
- **小屏幕（<992px）**：自动切换为垂直布局，字段选择器在筛选器下方

### 3. 筛选字段布局优化

- **超大屏幕（≥1200px）**：每行显示 6 个筛选字段（col-xl-2）
- **大屏幕（≥992px）**：每行显示 4 个筛选字段（col-lg-3）
- **中等屏幕（≥768px）**：每行显示 3 个筛选字段（col-md-4）
- **小屏幕（≥576px）**：每行显示 2 个筛选字段（col-sm-6）

### 4. 样式优化

添加了以下CSS样式：

```css
/* 平行布局样式 */
.parallel-layout {
    display: flex;
    gap: 1rem;
}

.parallel-layout .filter-section {
    flex: 2;
}

.parallel-layout .field-selector-section {
    flex: 1;
    min-height: 400px;
}

/* 字段选择器高度自适应 */
.field-selector-card {
    height: 100%;
    display: flex;
    flex-direction: column;
}

.field-selector-card .card-body {
    flex: 1;
    overflow-y: auto;
}

/* 响应式布局 */
@media (max-width: 991.98px) {
    .parallel-layout {
        flex-direction: column;
    }
    
    .parallel-layout .field-selector-section {
        min-height: auto;
    }
}

/* 水平布局拉伸优化 */
@media (min-width: 1200px) {
    .container-fluid {
        max-width: 100%;
        padding-left: 2rem;
        padding-right: 2rem;
    }
    
    #filterCard .card-body {
        padding: 2rem;
    }
    
    #filterCard .row {
        margin: 0 -0.75rem;
    }
    
    #filterCard .col-xl-2 {
        padding: 0 0.75rem;
    }
}

/* 超宽屏幕优化 */
@media (min-width: 1400px) {
    .container-fluid {
        padding-left: 3rem;
        padding-right: 3rem;
    }
    
    #filterCard .card-body {
        padding: 2.5rem;
    }
}
```

### 5. 字段选择器内部布局调整

- 将操作按钮从右侧列移到下方
- 优化按钮排列，使用 `d-grid gap-2` 实现垂直堆叠
- 保持所有功能按钮的完整性和可用性

## 功能保持

✅ **所有原有功能完全保留**：
- 筛选器的所有筛选条件
- 字段选择器的所有操作按钮
- 数据表格的显示和交互
- 响应式设计

## 优势

1. **空间利用更高效**：水平布局充分利用屏幕宽度，筛选器占据更多空间
2. **操作更便捷**：筛选和字段选择可以同时进行
3. **视觉层次更清晰**：两个功能模块并排显示，层次分明
4. **响应式友好**：在不同屏幕尺寸下都有最佳显示效果
5. **筛选效率更高**：每行显示更多筛选字段，减少滚动操作
6. **超宽屏幕优化**：在1400px+屏幕上进一步拉伸，充分利用大屏幕空间
